### What is the purpose of this software ?

With this software, you can easily select your application patterns at computer startup.

When you have a specific intention when starting your computer, you can get into it very fast by just selecting your mood.

Ex : Programming : IDE, github / Chill : chat app and browser.

### Required

* Python 3
* [`curses`](https://docs.python.org/3/library/curses.html) module (available on **UNIX** system by default).

### Launch and configure

You can launch main.py to test the script :
    `python3 main.py`
    
You can configure the softwares pattern in the config.py file, syntax in the example file.
